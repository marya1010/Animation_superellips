Кривая для суперэллипса задается параметрическими уравнениями:

x(t) = +- a cos^(2/n)t
y(t) = +- b sin^(2/n)t

, 0 <= t <= pi/2 и каждый +- может быть выбран отдельно, так что каждое значение
t дает четыре точки на кривой.

Эквивалентно, если t находится в диапазоне 0 <= t <= 2pi, то

x(t) = |cos t|^(2/n) * a sgn(cos t)
y(t) = |sin t|^(2/n) * b sgn(sin t)
